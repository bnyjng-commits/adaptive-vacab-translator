## 6. v0.1 완료를 위해 아직 남은 일 (배포 전 필수)

### Supabase에서 직접 해야 할 작업

**① students 테이블 생성**

Supabase 대시보드 → SQL Editor에서 아래 SQL 실행:

```sql
create table students (
  id uuid primary key references auth.users(id) on delete cascade,
  name text not null,
  role text not null default 'student',
  current_stage int not null default 1,
  session_count int not null default 0,
  stage_session_count int not null default 0,
  updated_at timestamptz not null default now()
);

alter table students enable row level security;

create policy "본인 데이터 조회" on students for select using (auth.uid() = id);
create policy "본인 데이터 수정" on students for update using (auth.uid() = id);
```

**② 테스트 계정 생성**

Supabase 대시보드 → Authentication → Users → "Invite user"로 학생 이메일 등록.
등록 후 students 테이블에 같은 UUID로 행 삽입:

```sql
insert into students (id, name) values ('여기에UUID', '홍길동');
```

### Vercel에서 직접 해야 할 작업

**③ 환경변수 등록**

Vercel 대시보드 → Project → Settings → Environment Variables:

| 변수명 | 값 | 환경 |
|--------|-----|------|
| `SUPABASE_URL` | `https://xxxxx.supabase.co` | Production, Preview, Development |
| `SUPABASE_ANON_KEY` | `eyJxxx...` | Production, Preview, Development |

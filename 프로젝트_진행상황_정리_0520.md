# 중도 입국 학생을 위한 적응형 교과 용어 번역 서비스
## 프로젝트 진행 상황 정리 (2026년 5월 21일 업데이트)

---

## 1. 프로젝트 개요

- **과목**: AI 교육서비스 모델 설계 (부산대학교 교육대학원 AI융합전공)
- **담당교수**: 남윤경, 조윤석 교수님
- **목표**: 중도 입국 학생(다문화 배경)이 수학 수업 중 교과서를 보면서 모르는 용어를 클릭하면 모국어 번역 툴팁이 뜨는 웹 앱 프로토타입 제작
- **대상 단원**: 중학교 1학년 수학 Ⅳ. 기본도형
- **배포 주소**: `adaptive-vacab-translator.vercel.app`
- **GitHub**: `https://github.com/bnyjng-commits/adaptive-vacab-translator`
- **사용 환경**: 학생은 태블릿 브라우저로 접속, 개발은 VS Code + Claude Code

## 4. Supabase 테이블 구조

### students 테이블

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| `id` | uuid | 기본키, Supabase Auth user.id와 연결 |
| `name` | text | 학생 이름 (default: '') |
| `role` | text | 역할 |
| `current_stage` | int4 | 현재 번역 단계 (1~4) |
| `session_count` | int4 | 누적 수업 차시 |
| `stage_session_count` | int4 | 현재 단계에서의 차시 수 |
| `last_attended_date` | text | 마지막 수업 날짜 (YYYY-MM-DD). 로그인 시 오늘과 비교해 차시 자동 카운트 |
| `updated_at` | timestamp | 마지막 업데이트 시각 |

> ⚠️ 주의: 코드에서 `user_id` → `id`, `stage` → `current_stage` 로 사용해야 함

### session_notes 테이블 ✅ 적용 완료

수업 종료 시 학생이 입력한 노트를 저장하는 테이블.

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| `id` | bigint (auto) | 기본키 |
| `student_id` | uuid | students.id 참조 |
| `date` | text | 수업 날짜 (YYYY-MM-DD) |
| `note_text` | text | 학생이 입력한 노트 |
| `stage` | int | 노트 작성 시점의 번역 단계 |
| `created_at` | timestamp | 생성 시각 |

## 5. Supabase RLS 정책 (적용 완료)

```sql
-- students 테이블
create policy "본인 조회" on public.students for select using (auth.uid() = id);
create policy "본인 삽입" on public.students for insert with check (auth.uid() = id);
create policy "본인 수정" on public.students for update using (auth.uid() = id);

-- session_notes 테이블
create policy "본인 삽입" on public.session_notes for insert with check (auth.uid() = student_id);
create policy "본인 조회" on public.session_notes for select using (auth.uid() = student_id);
```

# 작업지시서 — 적응형 교과 용어 번역 서비스
> Claude Code에 전달하는 전체 개발 명세서입니다.
> 마일스톤 하나가 완전히 작동하는 것을 확인한 후 다음으로 넘어갑니다.

---

## 프로젝트 한 줄 요약

중도 입국 학생이 수학 수업 중 태블릿으로 교과서를 보면서 모르는 용어를 클릭하면
모국어 번역 툴팁이 뜨는 웹앱. 번역 단계를 4단계로 조절할 수 있고, 차시가 쌓이면
자동으로 한국어 비중이 늘어나는 적응형 구조.

---

## MVP — v0.1: 로그인 + 단계 저장

**목표:** 학생이 로그인하면 자신의 번역 단계가 저장되고, 다음에 접속해도 유지된다.

### 기능 명세

**로그인 화면**
- 아이디(이메일) + 비밀번호 입력 폼
- 로그인 버튼 → Supabase Auth 인증
- 최초 1회 로그인 후 자동로그인 유지 (Supabase 세션 토큰)
- 로그아웃 버튼 (메인 화면 상단)
- 로그인 실패 시 안내 문구 표시

**데이터 저장 (Supabase)**
- 로그인 성공 시 학생 정보 불러오기
- 단계 변경 시 즉시 Supabase에 저장
- 다음 접속 때 저장된 단계로 자동 복원

**Supabase 테이블: `students`**

| 컬럼 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `id` | uuid | — | Supabase Auth UID (자동) |
| `name` | text | — | 학생 이름 |
| `role` | text | `'student'` | `'student'` 또는 `'teacher'` |
| `current_stage` | int | `1` | 현재 번역 단계 (1~4) |
| `session_count` | int | `0` | 누적 차시 수 |
| `stage_session_count` | int | `0` | 현재 단계 연속 차시 수 |
| `updated_at` | timestamp | now() | 마지막 업데이트 |

**완료 기준**
- [ ] 로그인 후 새로고침해도 로그인 상태 유지
- [ ] 단계 변경 후 새로고침해도 같은 단계로 복원
- [ ] Vercel 배포 후 실제 태블릿에서 확인

## v0.2: 자동 페이딩 로직

**목표:** 같은 단계에서 3차시 연속이면 4차시 시작 시 자동으로 다음 단계로 이동한다.

### 기능 명세

**차시 시작 버튼**
- 메인 화면에 "수업 시작" 버튼 추가
- 누르면 `session_count` +1, `stage_session_count` +1
- Supabase에 즉시 저장

**자동 전환 조건**
- 수업 시작 시 `stage_session_count >= 3` 이면 → `current_stage` +1, `stage_session_count` = 0으로 초기화
- 전환 시 안내 메시지: "이번 차시부터 [n]단계로 올라갑니다! 🎉"
- 4단계(번역 없음)에서는 더 이상 자동 전환 없음

**수동 변경**
- 학생이 단계를 직접 바꾸면 → `stage_session_count` = 0 초기화
- 어느 방향으로든 자유롭게 이동 가능 (1단계로 내려가는 것도 허용)

**완료 기준**
- [ ] 테스트 계정으로 3차시 진행 후 4차시 시작 시 자동으로 단계 상승 확인
- [ ] 수동 변경 시 카운터 초기화 확인

## v0.3: PDF 업로드 + 텍스트 레이어

**목표:** 교과서 PDF를 업로드하면 이미지로 표시되고, 용어를 클릭하면 툴팁이 뜬다.

### 기능 명세

**PDF 렌더링**
- PDF 업로드 버튼 (파일 선택)
- pdf.js(Mozilla CDN)로 페이지별 이미지 렌더링
- 페이지 이동 버튼 (이전 / 다음)

**텍스트 레이어**
- PDF에서 텍스트와 좌표 추출
- 원본 이미지 위에 투명한 텍스트 span 오버레이
- `math_vocab_geometry.json`에 있는 용어 → 점선 밑줄 표시
- 클릭 시 툴팁 표시 (기존 툴팁 로직 재사용)

**번역 단계 적용**
- 1단계: 모든 텍스트를 번역어로 표시
- 2단계: 수학 용어만 한국어 유지, 나머지 번역
- 3단계: 수학 용어만 번역어 표시, 나머지 한국어
- 4단계: 전체 한국어 원문

**완료 기준**
- [ ] 교과서 PDF 1페이지 업로드 후 용어 클릭 시 툴팁 표시 확인
- [ ] 단계 변경 시 텍스트 레이어 즉시 반영 확인

## v0.4: Claude API 연결

**목표:** 용어집에 없는 단어가 클릭되면 Claude API를 호출해 번역을 가져온다.

### 기능 명세

**2층위 번역 로직**
```
클릭한 단어
  ↓
math_vocab_geometry.json에 있음? → YES → 즉시 반환 (API 호출 없음)
  ↓ NO
Claude API 호출 → 번역 + 정의 반환 → 툴팁 표시
```

**API 호출 설정**
- 모델: `claude-sonnet-4-20250514`
- max_tokens: `200`
- 시스템 프롬프트 (고정):
```
너는 중학교 수학 교과서 번역 전문가야.
지금 텍스트는 중1 수학 기본도형 단원이야.
요청받은 한국어 단어의 번역어와 한 줄 정의를 아래 JSON 형식으로만 반환해.
다른 말은 절대 하지 마.
{"translation": "번역어", "definition": "정의 한 줄"}
```
- API 키: Vercel 환경변수 `ANTHROPIC_API_KEY`에서 읽음 (코드에 직접 쓰지 않음)

**UX**
- 로딩 중 툴팁에 스피너 표시
- 호출 실패 시: "번역을 불러올 수 없습니다" 표시

**완료 기준**
- [ ] 용어집에 없는 단어 클릭 시 3초 이내 번역 표시 확인
- [ ] API 키가 코드에 노출되지 않음 확인
